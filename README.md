## Fitur

Bagian ini merangkum fitur utama aplikasi, seperti:

- **Menambah tugas baru**
- **Melihat semua tugas**
- **Menghapus tugas yang sudah selesai**
- **Menyimpan data secara persisten menggunakan format JSON**
